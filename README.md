# weddingway

SETUP

Requirements:

- NodeJS
  - Install from: https://nodejs.org/en/
- NPM
  - Install from https://www.npmjs.com/
- Grunt Client (Only for developers)
  - https://gruntjs.com/getting-started
  
Running the App
- $npm install
- $node server

Optional
- grunt build
  - packages css
